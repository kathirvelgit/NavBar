# 🧭 Responsive Navigation Bar

A simple and modern Navigation Bar built using **HTML5** and **CSS3**. This project demonstrates the use of Flexbox for creating a clean horizontal navigation layout with a logo, navigation links, and a login button.

---

## 🚀 Features

✅ Modern Navigation Bar Layout

✅ HTML5 Semantic Elements

✅ CSS Flexbox Alignment

✅ Responsive Navigation Structure

✅ Logo Section

✅ Navigation Links

✅ Login Button

✅ Clean and Beginner-Friendly Design

---

## 🛠️ Technologies Used

* 🌐 HTML5
* 🎨 CSS3
* 📐 Flexbox
* 💻 Visual Studio Code
